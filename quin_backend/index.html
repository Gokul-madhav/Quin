Backend API

About

Node.js + Express APIs for Agora voice calls, QR code generation, and web-to-mobile call routing. Deployed on Vercel.

API Endpoints

  • GET /api/health
  • POST /api/agora/token
  • POST /api/agora/call/create
  • POST /api/agora/call/join
  • POST /api/qrcode/generate
  • POST /api/call/create-with-qr
  • POST /api/call/join-by-code
